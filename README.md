# 🚀 User Management System

## 📖 Descrição do Projeto

O **User Management System** é uma aplicação desenvolvida em Java com o objetivo de gerenciar funcionários por meio de um sistema simples em console.

O sistema permite realizar operações básicas como cadastro, listagem, busca e aplicação de aumento salarial, simulando um ambiente real de gerenciamento de dados.

Este projeto foi desenvolvido com foco em **boas práticas de programação orientada a objetos**, organização em camadas e tratamento de exceções.

---

## ⚙️ Funcionalidades

* ✔ Cadastro de usuários (Assalariado e Horista)
* ✔ Listagem de usuários
* ✔ Busca por nome
* ✔ Remoção de usuários
* ✔ Aplicação de aumento salarial para todos os usuários
* ✔ Tratamento de exceções para entradas inválidas
